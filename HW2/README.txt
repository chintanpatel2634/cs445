HW2: CS 445

Following should be installed in your computer to run this project.
1) Java Development Kit 11:
Link: https://www.oracle.com/technetwork/java/javase/downloads/jdk11-downloads-5066655.html
2) Maven (To build the project):
Link: https://linuxize.com/post/how-to-install-apache-maven-on-ubuntu-18-04/

Process to build the program:
1) After installing Java and Maven, download my project file, and unzip the file.
2) For this assignment, you can build problem-4 and problem-5 folder on the terminal.
3) Run "mvn clean" to clean the artifacts created in prior build.
4) Run "mvn install" to install the package in your local repository, so that you can use the dependencies.
5) Run "mvn test" to test the code.
